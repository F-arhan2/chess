# Chess Game in Java ♟️

## Overview

This project is a fully functional Chess Game developed entirely in Java. The game implements standard chess rules and provides an interactive interface for two players to play chess on a virtual board.

## Features

* Complete 8x8 chess board
* All standard chess pieces implemented
* Valid move checking for every piece
* Turn-based gameplay
* Capture opponent pieces
* Check and Checkmate detection
* Simple and user-friendly interface
* Object-Oriented Programming (OOP) design

## Technologies Used

* Java
* Java Swing/AWT (if used for GUI)
* Object-Oriented Programming Concepts

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/chess-game-java.git
```

2. Navigate to the project folder:

```bash
cd chess-game-java
```

3. Compile the project:

```bash
javac *.java
```

4. Run the game:

```bash
java Main
```
